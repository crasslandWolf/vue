## install

```javascript
npm install pzvue-select
```

## Attribute

参数          | 说明          | 类型       | 可选值         | 默认值
----------- | ----------- | -------- | ----------- | ----------------------------
placeholder | 占位符 | string   |             |
options     |    选项         | Array      |             | [{text: "不限",value: 0,default: true}]
default     |      默认值       | all   |             |
disabled    | 禁用          | boolean  | true, false | false
change      | change事件    | function |             | function(value){}
